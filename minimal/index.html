About Cambodia Monitor

Cambodia Monitor v2.0 is a free, real-time intelligence dashboard focused exclusively on Cambodia. It aggregates news from local and international sources, provides live weather and economic data, and monitors key locations.


Local Sources: Khmer Times, Phnom Penh Post, VOA Cambodia, Cambodianess

International Sources: Reuters, Al Jazeera, BBC, AP News, Nikkei Asia (filtered for Cambodia mentions)


Economic Data: World Bank API (GDP, inflation, FDI, trade, demographics)

Weather: Open-Meteo API (4 cities, real-time)

Exchange Rates: open.er-api.com (USD, THB, CNY, EUR, JPY to KHR)


Focus Areas: Cambodia, Phnom Penh, Siem Reap, Sihanoukville, Hun Manet, ASEAN, Mekong Region


All APIs are free and require no keys. Built with Leaflet.js and curated RSS feeds.
